# MKAppTool | macOS dSYM, xcarchive & LinkMap Analyzer

macOS desktop app for iOS developers: symbolicate crash addresses with dSYM/xcarchive, and analyze app binary size with LinkMap files.

Keywords: dsym tool mac, linkmap analyzer, xcarchive crash locate, ios symbolication, app size analysis

## Quick Start

Download [MKAppTool.pkg](https://github.com/mythkiven/mkBox/releases/download/MKAppTool/MKAppTool.pkg), open a `.xcarchive` or LinkMap `.txt`, paste crash addresses to locate source lines.
